# Source review checklist

- [ ] domeniu și publisher oficial confirmate;
- [ ] autoritatea și competența emitentului identificate;
- [ ] data publicării/actului și intervalul de aplicare confirmate;
- [ ] snapshot salvat legal, hash SHA-256 și locator stabil;
- [ ] claim-urile sunt atomice și citatele nu depășesc ce demonstrează;
- [ ] rang juridic, teritoriu și populație aplicabilă completate;
- [ ] contradicțiile și documentele superseded identificate;
- [ ] facts, gates, requirements și deadlines se leagă numai de claim-uri relevante;
- [ ] review due/hard expiry stabilite conform clasei de prospețime;
- [ ] reviewer diferit de autor pentru reguli critice;
- [ ] simularea golden cases nu produce regresii;
- [ ] canalul oficial are status tehnic onest (`SOURCE_ONLY`, `DEEP_LINK`, `ACTIVE`).
