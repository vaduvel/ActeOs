# PRD - LifeOS Romania v1

## Problema
Romanii nu se blocheaza doar pentru ca nu gasesc formularul. Se blocheaza pentru ca nu stiu traseul complet, ordinea pasilor, documentele conditionale, termenele si capcanele locale.

## Public tinta R1
- Persoane care se muta.
- Persoane care cumpara/vand masina.
- Persoane cu acte pierdute/expirate.
- Familii care au copil/elev.
- Antreprenori mici, in release-uri ulterioare.

## Jobs to be Done
- Cand mi se intampla un eveniment administrativ, vreau sa stiu tot ce trebuie sa fac ca sa nu ma plimb degeaba.
- Cand am documente de pregatit, vreau sa stiu ce lipseste/gresit inainte sa merg la institutie.
- Cand se schimba o regula sau expira un act, vreau sa fiu avertizat.

## Scope R1
- Event taxonomy si search natural language.
- Life event sessions.
- Event -> obligations -> journeys -> steps.
- Demo-safe flows: `life.moved`, `life.bought_vehicle`, `life.lost_documents`, `life.local_taxes_certificate`.
- Document readiness checks cu fixtures fake-safe.
- Curator portal minimal.
- Mobile UX pentru Home, Triage, Event Map, Route, Dossier.

## Non-goals R1
- Depunere oficiala in numele utilizatorului.
- Decizii juridice.
- Reguli reale neverificate pentru toate orasele.
- Marketplace de agentii.
