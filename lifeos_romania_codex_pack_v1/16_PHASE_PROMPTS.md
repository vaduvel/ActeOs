Create monorepo, tooling, Docker, healthchecks.

Implement JSON schemas and OpenAPI. Generate shared types.

Implement migrations from `10_DATABASE_SCHEMA.sql`; seed event taxonomy.

Implement predicate evaluator, ternary logic, obligation expansion, route hash.

Implement simple keyword/fuzzy resolver and required facts. No admin decisions.

Implement event sessions, facts update, resolve route, fetch route.

## P6 - Mobile
Implement Home, Triage, Event Map, Route, Dossier screens.

Implement classify/check interfaces with fake-safe fixtures.

## P8 - Curator
Implement sources, claims, rules, bundle publish/rollback.

## P9 - Notifications
Implement deadlines/expiry workers and notification model.

## P10 - Partners
Implement optional partner offers with neutrality tests.

## P11 - Observability/Security
