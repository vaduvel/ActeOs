# Codex Master Prompt - LifeOS Romania

You are building LifeOS Romania from this execution pack.

## Mission
Build a cross-platform mobile app and backend that turns life events into deterministic administrative routes.

## Non-negotiables
- Do not invent legal/admin rules.
- Demo data must be marked `demo_mode: true`.
- Critical production rules require source claims.
- LLM/AI may classify text or draft content offline, but never decides runtime eligibility or required documents.
- Deterministic rule engine is source of runtime decisions.
- Partner offers must not modify official routes.

## Stack
- React Native Expo + TypeScript for mobile.
- FastAPI + Python for API.
- Postgres for persistence.
- Optional Next.js/React curator portal.

## Execution
Follow `16_PHASE_PROMPTS.md` phase by phase. At the end of each phase, update implementation status and run tests.
